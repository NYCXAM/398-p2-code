## Checkpoint 2:  
**Commands used:**  
1. git log --oneline  
2. git revert 66f6716  
3. git push origin feature/119823026  
   
**Learnings:**  
Learned how to use revert to undo changes while keep the original commit.   
If I didn't learn this command, I'll probably just manually delete and commit  
the code. But this command prevent mistakes like accidentally deleting extra codes  
when deleted the bad code.

## Checkpoint 5:
**Commands used:**
1. git remote add upstream https://github.com/mdurrani808/git_project_1
2. git fetch upstream
3. git checkout main
4. git reset --hard upstream/main
5. git checkout feature/119823026
6. git rebase main
7. git push --force-with-lease origin feature/119823026

**Learnings:**
Learned how to sync my code with updated upstream code. This is useful when I'm working
on some forked repo and someone added new feature on the upstream version.
